# Maximizing_Clinical_Yield_Analysis
This repository contains an analyzed healthcare dataset and a technical report examining patient admissions, medication usage, disease prevalence and billing outcomes. The analysis highlights chronic conditions such as arthritis and diabetes and their impact on hospital capacity and clinical yield

Project Focus

The analysis examines:
- Patient admission patterns across admission types
- Common medical conditions, with emphasis on chronic diseases such as arthritis and diabetes
- Medication usage trends
- Billing outcomes and revenue performance
The goal is to understand how disease burden and operational flow influence hospital capacity and financial outcomes.

Contents of the Repository
- Analyzed Dataset: Cleaned and structured data used for analysis
- Technical Report: A detailed Excel-based analytical report covering:
- Story of Data
- Data Splitting and Preprocessing
- Pre-Analysis, In-Analysis, and Post-Analysis
- Visualizations, Insights, and Recommendations
- Tools Used
- Microsoft Excel
- Pivot Tables
- Grouping and aggregation
- Dashboards and charts

Key Insights Highlighted
- Chronic conditions such as arthritis and diabetes drive sustained admission volume
- Admission types influence hospital capacity planning
- Medication demand reflects long-term disease management needs
- Billing performance varies across hospital entities

Intended Use

This repository is intended for:
- Healthcare data analytics portfolios
- Demonstrating Excel-based analytical reporting
- Review by recruiters, educators, or analysts interested in healthcare operations and finance
